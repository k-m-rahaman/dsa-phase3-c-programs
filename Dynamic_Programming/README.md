# 📊 Dynamic Programming (Phase 3)

Dynamic Programming is used to solve complex problems by breaking them into smaller overlapping subproblems.

## 📌 Topics Covered

- Fibonacci (Memoization & Tabulation)
- Climbing Stairs
- Coin Change Problem
- 0/1 Knapsack
- Longest Common Subsequence (LCS)
- Longest Increasing Subsequence (LIS)

## 🎯 Key Learnings

- Memoization vs Tabulation
- Optimization of recursion
- Overlapping subproblems
- State transition thinking

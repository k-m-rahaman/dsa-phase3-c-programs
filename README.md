## 📌 Topics Covered

---

### 🔁 Backtracking

Backtracking is used to solve problems by exploring all possible solutions and undoing choices when necessary.

**Programs:**

* Subsets (Power Set)
* Permutations
* N-Queens Problem
* Rat in a Maze
* Combination Sum

---

### 🌳 Advanced Trees

Advanced operations on trees for optimized searching and traversal.

**Programs:**

* Lowest Common Ancestor (LCA)
* Diameter of Tree
* Check Balanced Tree
* Zig-Zag Traversal
* Boundary Traversal
* BST to Sorted Array

---

### 🌐 Advanced Graphs

Graph algorithms are crucial for solving real-world network problems.

**Programs:**

* Cycle Detection (DFS & BFS)
* Topological Sort (DFS & Kahn’s Algorithm)
* Dijkstra’s Algorithm
* Shortest Path (Unweighted Graph)
* Connected Components

---

### 📊 Dynamic Programming (DP)

DP is used to optimize recursive solutions by storing results.

**Programs:**

* Fibonacci (Memoization & Tabulation)
* Climbing Stairs
* Coin Change
* 0/1 Knapsack
* Longest Common Subsequence (LCS)
* Longest Increasing Subsequence (LIS)

---

### 🔍 Binary Search (Advanced)

Efficient searching techniques in sorted data.

**Programs:**

* Binary Search (Iterative & Recursive)
* Search in Rotated Sorted Array
* First and Last Occurrence
* Square Root using Binary Search

---

### 🧠 Greedy Algorithms

Greedy algorithms make locally optimal choices for global optimization.

**Programs:**

* Activity Selection
* Fractional Knapsack
* Job Sequencing Problem
